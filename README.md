## 2. Tipografía (Typography)

Para equilibrar lo técnico con lo profesional:

* **Títulos (Display):** `Inter` o `Montserrat` (Sans Serif).
    * *Uso:* Nombres y encabezados de sección. Transmite modernidad y claridad.
* **Cuerpo y Datos:** `Roboto` o `System UI`.
    * *Uso:* Extractos y párrafos de experiencia.
* **Acentos de Código:** `'JetBrains Mono', monospace`.
    * *Uso:* Tags, habilidades técnicas, y pequeños fragmentos de código. Da el look "Dev".

---

## 3. Bordes y Sombras (UI Elements)

* **Bordes de Card:** `1px solid #30363D` (Gris oscuro sutil).
* **Borde de Acento:** `border-left: 4px solid #FF9900` (En cards de proyectos destacados).
* **Radio de Esquina (Border-radius):** `12px` para cards, `50px` para botones pill. Un balance entre lo cuadrado (técnico) y lo redondeado (creativo).
* **Sombras (Glow Effect):**
    * `box-shadow: 0 0 20px rgba(255, 153, 0, 0.1);` (Un brillo naranja muy tenue para resaltar elementos AWS).

---

## 4. Márgenes y Espaciado (Layout)

* **Padding Interno (Cards):** `1.5rem` (24px) para dar "aire" al contenido.
* **Margen entre Secciones:** `4rem` (64px) para separar claramente la experiencia de los proyectos.
* **Gutter (Grilla):** `1.5rem` estándar de Bootstrap 5.
